# wpm

A terminal typing speed test built with Rust.

![Preview](./preview.png)

## Installation

### From source

```sh
git clone https://github.com/HayesBarber/wpm.git
cd wpm
cargo install --path .
```

### Homebrew

```sh
brew install HayesBarber/tap/wpm
```

## Usage

```sh
wpm          # 25 words (default)
wpm 50       # 50 words
wpm 100      # 100 words (max)
wpm -h       # show help
```

**Controls:**
- `Tab` — restart
- `Ctrl+C` / `Esc` — quit
