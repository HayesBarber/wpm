# wpm

A terminal typing speed test built with Rust.

![Preview](./preview.png)

## Installation

```sh
cargo install --path .
```

## Usage

```sh
wpm          # 25 words (default)
wpm 50       # 50 words
wpm 100      # 100 words (max)
```

**Controls:**
- `Tab` — restart
- `Ctrl+C` / `Esc` — quit
